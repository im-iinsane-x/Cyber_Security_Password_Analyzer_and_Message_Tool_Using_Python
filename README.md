# Cyber_Security_Password_Analyzer_and_Message_Tool_Using_Python
A Python-based cybersecurity utility that analyzes password strength, encrypts and decrypts messages, reverses text using recursion, performs text analysis, and provides an interactive menu-driven interface.

Project Purpose

This project was created to explore Python programming through practical cybersecurity-related utilities. It combines multiple concepts—including recursion, string processing, decision-making, and functions—into a single interactive application that demonstrates both programming fundamentals and basic security concepts.

Note: The encryption implemented in this project is a simple educational character-shifting technique and is not intended for real-world secure communication.
